# ADR-0001: Монорепозиторий на Turborepo + pnpm

- Status: Accepted
- Date: 2026-06-27

## Context

UI, магазин, подписки и админка должны развиваться независимо и переиспользовать общие слои.
Возможны будущие приложения (лендинг, админка, Telegram mini app, мобильное).

## Decision

Использовать монорепозиторий: Turborepo для оркестрации задач и pnpm workspace для зависимостей.
Приложения в `apps/*`, переиспользуемые пакеты в `packages/*`.

## Consequences

- UI библиотека и общие слои живут отдельно от приложения.
- Простое добавление новых приложений поверх общих пакетов.
- Нужна дисциплина границ пакетов и публичных `exports`.

## Alternatives Considered

- Один Next.js проект без пакетов: проще старт, но мешает независимому развитию слоёв.
