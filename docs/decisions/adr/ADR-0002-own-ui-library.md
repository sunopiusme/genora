# ADR-0002: Собственная UI библиотека

- Status: Accepted
- Date: 2026-06-27

## Context

Фронтенд не должен быть просто набором внешних shadcn-компонентов. Нужен свой
дизайн-системный слой с владением исходниками.

## Decision

Создать пакет `@genora/ui` со слоями primitives / components / blocks. Доступность —
на Radix primitives, визуальный слой полностью свой, токены — в `@genora/design-tokens`.

## Consequences

- Узнаваемый продукт и полный контроль над UI.
- Storybook как витрина библиотеки.
- Больше работы по поддержке собственного слоя.

## Alternatives Considered

- Установка закрытого UI-пакета: быстрее, но без контроля над дизайном и исходниками.
