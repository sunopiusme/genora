# ADR-0006: Платежи через ЮKassa / CloudPayments

- Status: Accepted
- Date: 2026-06-27

## Context

Магазин ориентирован на пользователей в РФ; нужны локальные платёжные методы и приём карт.

## Decision

Использовать ЮKassa и/или CloudPayments как платёжные провайдеры. Интеграция — в
`features/checkout/api`, подтверждение оплаты — через webhooks в `app/api`.

## Consequences

- Поддержка локальных платёжных методов.
- Статусы заказа и подписки обновляются по webhook.
- Зависимость от внешних провайдеров и их режимов работы.

## Alternatives Considered

- Только зарубежные платёжные шлюзы: ограниченная доступность для целевой аудитории.
