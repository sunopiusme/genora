# Success Metrics

## Продуктовые метрики

- Конверсия посетитель -> оформленная подписка.
- Средний чек и доля повторных продлений.
- Отток подписчиков (churn).

## Технические метрики

- Core Web Vitals в пределах целевых порогов (LCP, INP, CLS).
- Доля успешных платежей в платёжном потоке.
- Время отклика Server Actions для ключевых операций.

## Инженерные метрики качества

- Прохождение `pnpm lint` и `pnpm typecheck` без исключений.
- Соответствие кода `standards/CODE_STANDARD.md`.
- Покрытие критических сценариев (auth, checkout) тестами.
